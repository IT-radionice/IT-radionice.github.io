Operacije nad brojevima

Poslednji primer (5/2) je vratio 2, a ne 2.5
Zašto?
  • Vrednost 5 je tipa int i vrednost 2 je tipa int
  • Obzirom da su obe vrednosti celobrojne, radi se celobrojnom deljenje
  • Ostatak pri deljenju se zanemaruje
  • U novijim verzijama programskog jezika Python u ovakvim slučajevima se radi automatska konverzija u float

Stringovi

Tekstualne vrednosti
Zadaju se između "" ili '', svejedno
Zašto nam nije dovoljno da imamo samo jedan način zadavanja stringova?

Stringovi


>>> x = "Hello World!"
>>> print x
Hello World!
>>> type(x)
<type 'str'>

Operacije nad stringovima

+ je spajanje stringova (konkatenacija)
* je ponavljanje stringova

>>> "abcd"+"efgh"
'abcdefgh'
>>> "abcd"*2
'abcdabcd'

Indeksiranje stringova

Pronalaženje karaktera na zadatoj poziciji
Pozicija se zadaje kao [indeks]

>>> x = "Hello World"
>>> x[0] #String počinje na poziciji 0
'H'
>>> x[4]
'o'
>>> x[-1] #Poslednji karakter se nalazi na poziciji -1
'd'
>>> x[-2]
'l'

Isecanje stringova

Pronalaženje podstring za zadati početak i kraj
Početak i kraj se zadaju kao [početak:kraj]

>>> x = "Hello World"
>>> x[4:7]
'o W'
>>> x[4:]
'o World'
>>> x[:6]
'Hello '

Funkcije nad stringovima

  • s.lower() – pretvara string s u mala slova
  • s.upper() – pretvara string s velika slova
  • s.split(separator) – deli string po separatoru
  • s.join(list) – spaja stringove iz liste po s
  • s.replace(oldsub, newsub) – u stringu s pretvori sve podstringove oldsub sa newsub

Funkcije nad stringovima


>>> s = "Lorem ipsum dolor sit amet, consectetur adipiscing elit."
>>> s.lower()
'lorem ipsum dolor sit amet, consectetur adipiscing elit.'
>>> s.upper()
'LOREM IPSUM DOLOR SIT AMET, CONSECTETUR ADIPISCING ELIT.'
>>> l=s.split(" ")
>>> l
['Lorem', 'ipsum', 'dolor', 'sit', 'amet,', 'consectetur', 'adipiscing', 
	'elit.']
>>> " ".join(l)
'Lorem ipsum dolor sit amet, consectetur adipiscing elit.'
>>> s.replace("s","-|-")
'Lorem ip-|-um dolor -|-it amet, con-|-ectetur adipi-|-cing elit.'

Liste

Nizovi proizvoljnih vrednosti koje mogu da budu različitog tipa

  • Mogu da se indeksiraju i isecaju kao i stringovi
  • Mogu da se spajaju i ponavljaju kao i stringovi

Liste


>>> l = [1,2,"Hello World", 3]
>>> l[0] #indeksiranje
1
>>> l[2]
'Hello World'
>>> l[-1]
3
>>> l[1:3] #isecanje
[2, 'Hello World']
>>> l+[4,5] #konkatenacija

>>> l*2 #ponavljanje

Dodavanje elementa u listu

  • append(e) - dodavanje elementa na kraj liste
  • insert(indeks, e) - umetanje elementa na zadatu poziciju

>>> l = [1,2,"Hello World", 3]
>>> l
[1, 2, 'Hello World', 3]
>>> l.append("dodati element")
>>> l
[1, 2, 'Hello World', 3, 'dodati element']
>>> l.insert(2,"insertovani element")
>>> l
[1, 2, 'insertovani element', 'Hello World', 3, 'dodati element']

Brisanje elementa iz liste

  • del l[i] - brisanje elementa na indeksu i iz liste l
  • remove(e) - brisanje prve pojave elementa e iz liste

>>> l
[1, 2, 'insertovani element', 'Hello World', 3, 'dodati element']
>>> del l[2]
>>> l
[1, 2, 'Hello World', 3, 'dodati element']
>>> l.remove("dodati element")
>>> l
[1, 2, 'Hello World', 3]

Izmena vrednosti elementa u listi

  • l[i]=nova_vrednost 

>>> l
[1, 2, 'Hello World', 3]
>>> l[3]="nova vrednost"
>>> l
[1, 2, 'Hello World', 'nova vrednost']

Grananje

Motivacija

Svi programi koje smo do sada pisali izvršavali su se linearno
Često postoji potreba da se deo programskog koda izvrši (ili ne izvrši) u zavisnosti od ispunjenosti nekog uslova

if


>>> x = 5
>>> if x > 0: # if <uslov>:
	print "broj x je pozitivan" # telo
	
broj x je pozitivan
  • Telo je zadato tako što je indentovano jedan stepen u odnosu na uslov
  • Dobra praksa (pisanje čistog koda) je deo sintakse programskog jezika Python

if ... else


>>> x = -3
>>> if x>0:
	print "x je pozitivan"
else:
	print "x nije pozitivan"

x nije pozitivan
  • else - kod koji će se izvršiti ukoliko uslov nije ispunjen

if ... elif ... else


>>> x = 0
>>> if x>0:
	print "x je pozitivan"
elif x==0:
	print "x je neutralan"
else:
	print "x je negativan"

x je neutralan
  • elif - može da se ponovi proizvoljan broj puta

Zadavanje uslova

Izraz čija vrednost može da bude True ili False
Zadaje se korišćenjem operatora poređenja
  • == (po čemu se razlikuje od =?)
  • !=
  • <
  • <=
  • >
  • >=

Petlje

Motivacija

Deo koda koji se više puta izvršava
  • Unapred zadat broj puta - for-petlja (konačne petlje)
  • Dok god važi neki uslov - while-petlja (uslovne petlje)

for petlja

Prolazak kroz kolekciju

>>> l=[1,3,"Hello World",5]
>>> for e in l: #u svakoj iteraciji e uzme vrednost jednog elementa iz liste
	print e

1
3
'Hello World'
5
Unapred znamo koliko puta će se izvršiti petlja

Zadavanje kolekcija

Možemo da navedemo sve elemente kolekcije

>>> l=[1,3,"Hello World",5]
Često nam treba kolekcija sa elementima u određenom opsegu
  • Na primer, celi brojevi od 0 do 10

Zadavanje kolekcija sa elementima u opsegu

  • range(kraj) - lista brojeva od 0 do kraj-1
  • range(pocetak, kraj[, korak]) - lista brojeva od pocetak do kraj-1 sa korakom korak

>>> range(5) #svi brojevi od 0 do 4
[0, 1, 2, 3, 4]
>>> range(2,7) #svi brojevi od 2 do 6
[2, 3, 4, 5, 6]
>>> range(0,10,2) #svi parni brojevi od 0 do 9
[0, 2, 4, 6, 8]
