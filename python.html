Funkcije nad stringovima

  • s.lower() – pretvara string s u mala slova
  • s.upper() – pretvara string s velika slova

  • s.join(list) – spaja stringove iz liste po s
  • s.replace(oldsub, newsub) – u stringu s pretvori sve podstringove oldsub sa newsub

Motivacija

Svi programi koje smo do sada pisali izvršavali su se linearno
Često postoji potreba da se deo programskog koda izvrši (ili ne izvrši) u zavisnosti od ispunjenosti nekog uslova

if


>>> x = 5
>>> if x > 0: # if <uslov>:
	print "broj x je pozitivan" # telo
	
broj x je pozitivan
  • Telo je zadato tako što je indentovano jedan stepen u odnosu na uslov
  • Dobra praksa (pisanje čistog koda) je deo sintakse programskog jezika Python

if ... else


>>> x = -3
>>> if x>0:
	print "x je pozitivan"
else:
	print "x nije pozitivan"

x nije pozitivan
  • else - kod koji će se izvršiti ukoliko uslov nije ispunjen

if ... elif ... else


>>> x = 0
>>> if x>0:
	print "x je pozitivan"
elif x==0:
	print "x je neutralan"
else:
	print "x je negativan"

x je neutralan
  • elif - može da se ponovi proizvoljan broj puta

Zadatak - iks oks za dva igrača

Iks oks

  • Iks oks (tic-tac-toe) je igra u kojoj dva igrača naizmenično upisuju X i O u matricu dimenzija 3x3
  • Pobeđuje onaj igrač koji horizontalno, vertikalno ili dijagonalno povežu 3 znaka
  • Napisati Python program koji omogućuje da dva igrača igraju iks oks

format table

  • Tabla je lista karaktera od 0 do 8
  • Inicijalno je prazna

def inicijalizuj_tablu():
	return [' ']*9

Raspored polja na tabli

  • Tabla se prikazuje u konzoli
   |   |
 6 | 7 | 8
   |   |
-----------
   |   |
 3 | 4 | 5
   |   |
-----------
   |   |
 0 | 1 | 2
   |   |

prikaz table

  • Tabla se prikazuje u konzoli

def prikazi_tablu(tabla):
	print '   |   |'
	print ' ' + tabla[6] + ' | ' + tabla[7] + ' | ' + tabla[8]
	print '   |   |'
	print '-----------'
	print '   |   |'
	print ' ' + tabla[3] + ' | ' + tabla[4] + ' | ' + tabla[5]
	print '   |   |'
	print '-----------'
	print '   |   |'
	print ' ' + tabla[0] + ' | ' + tabla[1] + ' | ' + tabla[2]
	print '   |   |'
	print ''

preuzimanje podataka sa tastature

  • Igrači unose poziciju u koju se upisuje karakter

def preuzmi_poziciju():
	ret_val = ""
	while not ret_val in "0 1 2 3 4 5 6 7 8".split(' '):
		ret_val = raw_input("pozicija: ")
	return int(ret_val)

upis podatka u tablu

  • Moguće je upisati podatak jedino ako je pozicija prazna

def je_slobodno_mesto(tabla,pozicija):
	return tabla[pozicija]==' '
  • Uneti karakter se upisuje u odgovarajuće mesto na tabli

def upisi_poziciju(tabla, pozicija, karakter):
	tabla[pozicija]=karakter

popunjena tabla

  • Ukoliko nema više slobodnih polja na tabli, igra je završena

def je_kraj(tabla):
	for pozicija in range(9):
		if je_slobodno_mesto(tabla,pozicija):
			return False
	return True

pobeda

  • Ukoliko je igrač povezao dva polja, pobedio je

def je_pobedio(tabla, znak):
	return ((tabla[6] == znak and tabla[7] == znak and tabla[8] == znak) or # gornja horizontala
		(tabla[3] == znak and tabla[4] == znak and tabla[5] == znak) or # srednja horizontala
		(tabla[0] == znak and tabla[1] == znak and tabla[2] == znak) or # donja horizontala
		(tabla[6] == znak and tabla[3] == znak and tabla[0] == znak) or # znakva vertikala
		(tabla[7] == znak and tabla[4] == znak and tabla[1] == znak) or # srednja vertikala
		(tabla[8] == znak and tabla[5] == znak and tabla[2] == znak) or # desna vertikala
		(tabla[6] == znak and tabla[4] == znak and tabla[2] == znak) or # dijagonala
		(tabla[8] == znak and tabla[4] == znak and tabla[0] == znak)) # dijagonala

Potez

  • Preuzima se pozicija sa tastatre
  • Karakter se upiše u tablu na tu poziciju
  • Proveri se da li je igra završena (pobeda ili popunjena tabla)

def potez(tabla, znak):
	print 'igra '+znak
	pozicija = preuzmi_poziciju() #preuzimanje pozicije
	#sve dok se ne unese slobodno mesto
	while not je_slobodno_mesto(tabla,pozicija): 
		pozicija = preuzmi_poziciju()
	upisi_poziciju(tabla, pozicija, znak) #upis karaktera na poziciju
	prikazi_tablu(tabla)
	if je_pobedio(tabla, znak): #pobeda
		print 'pobedio je '+znak
		return True
	if je_kraj(tabla): #popunjena tabla
		print 'nema pobednika'
		return True
	return False

Igra

  • Inicijalizuje se prazna tabla
  • Igrači igraju naizmenično dok neko ne pobedi
  • Proveri se da li je igra završena (pobeda ili popunjena tabla)

def igra():
	tabla = inicijalizuj_tablu() #prazna tabla
	prikazi_tablu(tabla)
	while True:
		kraj = potez(tabla,'x') #igra x
		if kraj:
			break
		kraj = potez(tabla,'o') #igra o
		if kraj:
			break
	print 'kraj'
