Tipovi podataka

Numerički tipovi podataka

  • Celi brojevi (int)
  • Razlomljeni brojevi (float)

Celi brojevi

Na primer:
  • 5
  • 0
  • -8
  • ...

>>> x = 5
>>> print x
5
>>> type(x)
<type 'int'>

Razlomljeni brojevi

Na primer:
  • 3.14
  • 2.72
  • -15.3333333
  • ...

>>> x = 3.14
>>> print x
3.14
>>> type(x)
<type 'float'>

Operacije nad brojevima

Poslednji primer (5/2) je vratio 2, a ne 2.5
Zašto?
  • Vrednost 5 je tipa int i vrednost 2 je tipa int
  • Obzirom da su obe vrednosti celobrojne, radi se celobrojnom deljenje
  • Ostatak pri deljenju se zanemaruje
  • U novijim verzijama programskog jezika Python u ovakvim slučajevima se radi automatska konverzija u float

Stringovi

Tekstualne vrednosti
Zadaju se između "" ili '', svejedno
Zašto nam nije dovoljno da imamo samo jedan način zadavanja stringova?

Stringovi


>>> x = "Hello World!"
>>> print x
Hello World!
>>> type(x)
<type 'str'>

Operacije nad stringovima

+ je spajanje stringova (konkatenacija)
* je ponavljanje stringova

>>> "abcd"+"efgh"
'abcdefgh'
>>> "abcd"*2
'abcdabcd'

Indeksiranje stringova

Pronalaženje karaktera na zadatoj poziciji
Pozicija se zadaje kao [indeks]

>>> x = "Hello World"
>>> x[0] #String počinje na poziciji 0
'H'
>>> x[4]
'o'
>>> x[-1] #Poslednji karakter se nalazi na poziciji -1
'd'
>>> x[-2]
'l'

Isecanje stringova

Pronalaženje podstring za zadati početak i kraj
Početak i kraj se zadaju kao [početak:kraj]

>>> x = "Hello World"
>>> x[4:7]
'o W'
>>> x[4:]
'o World'
>>> x[:6]
'Hello '

Funkcije nad stringovima

  • s.lower() – pretvara string s u mala slova
  • s.upper() – pretvara string s velika slova
  • s.split(separator) – deli string po separatoru
  • s.join(list) – spaja stringove iz liste po s
  • s.replace(oldsub, newsub) – u stringu s pretvori sve podstringove oldsub sa newsub
